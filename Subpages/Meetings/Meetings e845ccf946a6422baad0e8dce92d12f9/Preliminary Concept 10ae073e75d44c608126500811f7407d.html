Topic of Discussion

Concepting Notes

Main Idea

  • World boss spawns far away from players, who can build up defences and resources or skirmish boss along the way.

Fighting mechanics

  • Melee
  • Ranged
  • Structures

Lose Condition

  • Boss reaches heart of the village you’re defending
  • Will despawn and allow players to rebuild
    • Leave carnage untouched to preserve persistence

Win Condition

  • Defeating the boss provides lots of loot
    • XP/Level for personal gain
    • Spawns resource spawn
      • Either special resources spawn there
      • Or standard resources but lots more

Building

  • Use blueprint to place structures
  • Structures have HP, can be destroyed by enemies
  • Either via grid system or free-placing
  • Structures can be rotated
  • Structures have fixed size
  • Very simplistic
    • Place greyed out version and then throw resources at it to build it
      • x wood, x stone, etc.
  • Structures can be moved/destroyed by owner
    • If owner leaves, anyone can do it

Online Persistence

  • Player
    • Starts with some resources
    • Player level
      • Unlocks buildings
      • Boost(?) when unlocked everything
  • Server
    • Wipes when no one online
    • Resources respawn when boss respawns
      • Standard resources just respawn
      • Boss resources stay until depleted
    • Server has hub where everyone can store resources to share with others

Communication

  • Done through emotes/pictos
    • Emoji wheel
  • Pinging / pointing

Controls

  • Point & Click
    • Interact / Attack
    • Set destination
    • Continuous clicking
  • Keyboard
